
# CONTENTSHIFT FIVERR GALLERY SPECIFICATIONS
=============================================

You need to create 5 images for your Fiverr Gig gallery. 
Since automated image generation isn't available in this environment, 
you have these options:

OPTION 1: USE CANVA (RECOMMENDED - FREE)
1. Go to canva.com
2. Create a "Custom Size" design: 800x600px
3. Use the specifications below for each image
4. Export as PNG/JPG
5. Upload to Fiverr Gig gallery

OPTION 2: USE POWERPOINT/GOOGLE SLIDES
1. Create slides at 800x600px dimensions
2. Follow specifications below
3. Export each slide as PNG
4. Upload to Fiverr

OPTION 3: HIRE A DESIGNER ON FIVERR ($5-15)
Search: "simple service gallery images" or "infographic"

USAGE INSTRUCTIONS:
==================
1. Create these 5 images using your preferred method
2. Name them: 01_service_overview.png through 05_results.png
3. Go to your Fiverr Gig editor → "Gallery" tab
4. Upload all 5 images
5. Arrange in order: 1→2→3→4→5 for optimal storytelling flow

PRO TIPS:
=========
- Use consistent fonts: Bold for titles, regular for body
- Keep text readable at small sizes (minimum 18pt for body text)
- Include your logo/watermark subtly if desired
- Test how images look in Fiverr's small preview format
- Update images quarterly or when you refine your service
